# 移动端机台档案操作手册

## 1. 登录
- 输入账号密码，点击“登录”进入机台档案页面。

## 2. 流水号识别
- 点击拍照上传控件拍摄机台铭牌。
- 系统自动OCR识别流水号并填入搜索框。
- 若识别失败，可手动输入流水号并检索。

## 3. 搜索与选择机台
- 在搜索框输入机台编号/名称/规格/状态。
- 在结果列表点击目标机台，系统高亮选中。

## 4. 上传机台图片
- 在“已选机台”下方选择图片（最多5张）。
- 系统自动压缩到<=500KB后上传。
- 上传过程中显示实时进度；失败自动重试最多3次。

## 5. 注意事项
- 图片需清晰、避免反光和遮挡。
- OCR耗时通常在2秒内，弱网下可能略有波动。
- 设备离线时可查看最近50条缓存档案数据。
