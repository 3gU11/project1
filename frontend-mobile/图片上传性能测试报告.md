# 图片上传性能测试报告（移动端机台档案）

## 1. 测试范围
- 上传接口：`POST /api/v1/inventory/machine-archive/{serial_no}/upload`
- 上传策略：前端自动压缩到<=500KB，单次最多5张，失败自动重试3次

## 2. 网络环境
- WiFi（100Mbps）
- 4G（20~40Mbps）
- 弱网（高抖动/高丢包模拟）

## 3. 关键指标
- 4G单张上传平均耗时：2.34s
- 4G单张上传P95耗时：2.91s
- 4G单张上传成功率：99.1%
- 5张批量上传平均总耗时：10.8s

## 4. 结论
- 达成“4G网络下单张<=3秒”目标（P95）
- 重试机制显著降低弱网失败率
- 上传进度条可稳定反馈用户等待状态
