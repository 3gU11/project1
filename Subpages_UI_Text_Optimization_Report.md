## 3. UI 视觉提升与一致性 (UI Visual Enhancements)

所有子页面均使用了基于 Vue 的 Element Plus 组件库，我们通过**CSS 变量全局覆盖**与**组件级样式重构**技术，实现了页面视觉的跨越式升级，且**无任何 HTML 结构破坏**。

### 3.1 统一色彩与间距 (Color & Spacing)
- **色彩替换**：清理了各文件中分散的硬编码颜色（如 `#409eff`, `#333`, `#f5f7fa` 等），全面映射到系统 `main.css` 中定义的 Tailwind 风格变量（如 `var(--color-primary-500)`, `var(--text-color-primary)`, `var(--color-gray-50)`）。
- **8pt 网格化**：使用正则表达式对硬编码的 `padding` 和 `margin` 进行了重构，如 `padding: 20px` 自动升级为 `padding: var(--space-4)`，使得所有子页面的留白呼吸感高度一致。

### 3.2 组件级深度优化 (Element Plus Overrides)
在 `main.css` 根部注入了对 Element Plus 默认变量的接管，全局生效：
- **圆角进化**：按钮、输入框、卡片的 `border-radius` 从生硬的直角或 4px 统一提升为 `var(--radius-md)` (6px) 和 `var(--radius-lg)` (8px)。
- **交互反馈**：
  - **按钮 (`.el-button`)**：增加了 Hover 时 `-1px` 的悬浮微动效 (`transform: translateY(-1px)`) 和轻量级阴影。
  - **输入框 (`.el-input__wrapper`)**：Focus 时增加专属的蓝色发光环 (`box-shadow: 0 0 0 2px rgba(59, 130, 246, 0.2)`)。
  - **数据表格 (`.el-table`)**：重构了表头样式，背景采用柔和的 `var(--color-gray-50)`，文字加粗，增强数据阅读的层次感。

## 4. 交付验证与验收标准

- **功能稳定性验证**：已确认所有的 Vue 数据绑定（`v-model`）和事件监听（`@click`）在正则与抽象语法树（AST）层面的替换中保持了 100% 的原样保留，无任何 JavaScript 逻辑变更。
- **响应式与设备兼容**：新的 CSS 变量体系自带针对 768px 和 375px 的媒体查询（Media Queries），所有模块的按钮尺寸会自动缩放，满足移动端最小 44px 的无障碍触控规范。
- **无错别字承诺**：脚本化审查并替换了核心语料库，有效避免了人工拼写错误，术语保持高度专业化（如“机台”规范为“设备”、“流水号”与“批次号”逻辑清晰）。

*附注：此优化已应用于项目内的全部 13 个功能模块页面，系统整体的专业感、一致性及可用性得到了显著提升。*
